# Physical AI & Humanoid Robotics Course

Welcome to the Physical AI & Humanoid Robotics Course - a comprehensive educational resource covering modern robotics, AI, and humanoid development.

This course is designed to provide a complete learning path for developing humanoid robots with AI capabilities. It covers:

- **Module 1**: Robotic Nervous System (ROS 2)
- **Module 2**: The Digital Twin (Gazebo & Unity)
- **Module 3**: AI-Robot Brain (NVIDIA Isaac)
- **Module 4**: Vision-Language-Action (VLA) Models
- **Module 5**: RAG Chatbots for Robotics

## Documentation Website

The complete course materials are available as a Docusaurus-powered website. To run the documentation locally:

```bash
cd book
yarn install
yarn start
```

## Repository Structure

- `book/` - Docusaurus documentation website source
- `docs/` - Course content and modules
- `backend/` - Backend services for the course
- `frontend/` - Frontend components
- `specs/` - Feature specifications

## Deployment

This site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.
